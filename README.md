# containers
Это моя собственная реализация некоторых стандартных контейнеров на языке C++. Проект включает реализацию map (словарь), set (множество), vector (вектор) и multiset (мультимножество) с реализацией итераторов и красно-чёрного дерева, тесты c использованием библиотеки GTest и Makefile со стандартными целями.
